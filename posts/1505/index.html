Ubuntu折腾札记

 2016.11.25  2020.4.29  计算机相关  1972  4 分钟

大一期末，由于回家的火车票买得晚，所以需要在学校多留几日。其时，舍友都已回家去了。我在宿舍闲着无聊，开始折腾我的电脑了。那时候，距离Win10正式版的发布还有一个月左右，但网上已经有各种测试版了。于是，我把自己的电脑升级成了Win10。但这并没有满足我，我又开始搜索“除了Windows，电脑还能装什么系统”。霎那间，一些崭新的名词出现在了我的眼前，比如“Linux”、“Ubuntu“、”黑苹果“等等。也就是从那个时候起，我第一次知道了Linux。第一次装Ubuntu并没有费很大的力气，当桌面伴随着那独特的启动音展现在我的面前时，我情不自禁的哇了一声。倒不是说界面有多么华美，只是面对全新的完全不同于Windows的桌面时，我真的很惊奇。然而，开始使用时却是步步维艰，几乎不管做什么都得一边百度一边操作。终于，新奇感过后，我在Windows下毫不犹豫的格式化了Ubuntu的安装盘。要是事情就此戛然而止也就不会有这篇文章出现了，恨就恨在自此以后，我一次又一次的安装Ubuntu，然后又删掉，循环往复，每次都要浪费半天时间。说到底，我终究不能忘了Linux。前几日，我又一次折腾了我的电脑。然而，这一次，我颇满意自己的创作，竟渐渐生发出就此告别Windows的想法。不管结果如何，我觉得有必要把我这一次的折腾记录下来，也方便日后查看。

安装

不具体记述每一步的操作，只记录一下分区情况。由于电脑中装有Windows,所以不能让Ubuntu自动安装。在某一步选择其他选项，然后开始分区。 1、第一次分区 上一步骤点“+”，进行如下设置： 挂载点：“/” 大小：22000MB 新分区的类型：主分区 新分区的位置：空间起始位置 用于：EXT4日志文件系统

2、第二次分区 “空闲”处，继续点“+”，如下设置， 挂载点：（不设置） 大小：2048MB 新分区的类型：逻辑分区 新分区的位置：空间起始位置 用于：交换空间

3、第三次分区 “空闲”处，继续点“+”，如下设置， 挂载点：/boot 大小：200MB 新分区的类型：逻辑分区 新分区的位置：空间起始位置 用于：EXT4日志文件系统 第三次分区完毕

4、第四次分区 “空闲”处，继续点“+”，如下设置， 挂载点：/home 大小：（剩余全部空间，剩下显示多少，就多少） 新分区的类型：逻辑分区 新分区的位置：空间起始位置 用于：EXT4日志文件系统

注意：前三次分区可以不按顺序，但如果电脑硬盘是mbr分区，则应把第一次分区放在第三步，因为mbr分区只能有四个主分区（好像是这样，记不清了）。推荐把分区类型改为gpt。

整理

装完之后要进行整理，我是按照csdn上的这篇博客整理的，贴在这："安装Ubuntu 16.04后要做的事 - skykingf的专栏 - 博客频道 - CSDN.NET"。当然，有的地方我并没有使用他的建议。为了能在状态栏显示网速，我用Indicator-SysMonitor代替了Syspeek。Indicator-SysMonitor的安装方法如下：

sudo add-apt-repository ppa:fossfreedom/indicator-sysmonitor
sudo apt-get update
sudo apt-get install indicator-sysmonitor

安装完毕以后，终端内输入indicator-sysmonitor即可启动，然后简单设置一下就能实现开机自启动。

装完WPS Office后，还需要解决一下字体缺失的问题，我参照了开源中国上面的这个帖子：WPS for Linux（ubuntu）字体配置(字体缺失解决办法) - renwofei423 - 开源中国社区。

Sublime Text 3的Ubuntu版本有一个特别大的缺憾，那就是不能输入中文，好在已经有大神提供了解决方案，并开源在了Github上：lyfeyaj/sublime-text-imfix: Fix Sublime Text 2/3 Input method problem under Ubuntu。

美化

其实换个壁纸就能把Ubuntu变得美美的，但还有更厉害的工具。参照这个帖子：ubuntu16.04主题美化和软件推荐 - ZeeCoder - 博客频道 - CSDN.NET。

软件

我使用的密码管理软件是KeePass，但是Linux下的KeePass在显示中文方面有问题，需要调教。第一步是到官网下载中文翻译包。然后把语言包放在/home/yourname/.local/share/KeePass文件夹下，如果没有就新建一个。然后在KeePass中选择view->change language，选择中文。别忙着重启软件，由于unix下强制使用系统字体，所以还需要去掉勾选“force using system font (unix only)”这个选项。位置在tools->options->interface最末尾。

笔记应用我选择的是Google Keep（Chrome应用），虽说它并没有OneNote般功能丰富，但可以满足我的要求。至于同步，也很简单，修改hosts就可以了。

写在最后

经过这样一番折腾，我的Ubuntu已经我完全满足了我的日常需求。最后，贴几张截图做个纪念。
